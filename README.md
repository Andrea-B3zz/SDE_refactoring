## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/HZ-OOP-2324/oop-team01.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
